Project-Webbutveckling-med-PHP
==============================

Installationsanvisningar:

+ Skapa en databas med filen <i>sql-create.sql</i> i mappen <i>applikation/sql</i>
+ Ladda in testdata med scriptet <i>sql-insert-test-data.sql</i> i mappen <i>applikation/sql</i>
+ Kopiera över filerna i mappen <i>applikation</i> till din www-katalog på servern
+ Ändra uppgifterna i filen <i>DatabaseSettings.php</i> i mappen <i>applikation/src/settings</i> så att uppgifterna speglar din egna databas
+ Nu är applikationen klar att köras
